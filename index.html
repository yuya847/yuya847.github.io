テキスト/ファイル ⇄ Base64 & 連続QRブリッジ

入力

ファイルをドラッグ＆ドロップ または
※ テキスト→Base64 / Base64→テキスト / Base64→ファイル で表示

出力

QRシーケンス出力

00:00 00:00 0 / 0

QR受信（カメラ / HID）→ 出力へ反映

※HTTPS/localhostで動作。ヘッダ有れば順序復元します。
HTA側のタイル数に合わせてください（例: 2×2）。
受信進捗（ヘッダ付き時）: 0/0

ヘルプ

概要

  • 医局のバーコードリーダーDS9308を使ってカルテに情報を送り込めるプログラムです。
  • ①テキスト/ファイルをBase64という文字列の変換を介してQRコード化します。
  • ②QRコードをカルテに繋いだバーコードリーダーにかざしましょう。入力したテキストは共有フォルダの鈴木のプログラムにある変換プログラムでBase64を文字列あるいはファイルに変換しましょう。
  • QRコードからカルテ内のファイル、データの受信も可能です。

モード

  • テキスト ↔ Base64：相互変換。Ctrl+Enterで変換。
  • ファイル → Base64：ドラッグ&ドロップでBase64化。
  • Base64 → ファイル：入力のBase64を「名前を付けて保存」で復元。
  • QR→Base64 / QR→テキスト：単一QRを連続受信。検出枠ガイド表示。
  • QR（複数）→Base64 / QR（複数）→テキスト：映像をグリッド分割して複数同時読み取り。

連続QR送出

  • QR表示（連続）で最大約700文字/枚に分割して送出します。
  • FPSは0.5〜120。スライダーまたは数値ボックスで指定可能。
  • ヘッダなし（EMR向け）／ヘッダあり（順序復元＆整合性チェック）。
